# 阈值与常量统一

## 2.1 写入门控

- 评分公式:
  - S_write = 0.35*novelty + 0.25*importance + 0.20*user_explicit + 0.20*confidence
- 领域阈值:
  - tau_write(d) = tau_0 + delta_d
- 默认常量:
  - tau_0 = 0.65
  - delta_education_core = -0.15
  - delta_general_default = 0.00
  - delta_casual_chat = +0.15
- 决策规则:
  - S_write >= tau_write(d) -> candidate_active
  - 0.50 <= S_write < tau_write(d) -> candidate_hold
  - S_write < 0.50 -> 丢弃

## 2.2 检索重排

- 评分公式:
  - S_recall = 0.35*semantic_sim + 0.20*recency + 0.20*importance + 0.15*confidence - 0.10*conflict_penalty
- 默认权重:
  - semantic_sim=0.35, recency=0.20, importance=0.20, confidence=0.15, conflict_penalty=0.10

## 2.3 冲突判别

- 判别公式:
  - Lambda = log(P(m_new)/P(m_old)) + SUM_j w_j * log(P(e_j|m_new)/P(e_j|m_old))
- 默认阈值:
  - tau_0 = 0.5
  - tau_1 = 2.0
- 五区间决策:
  - Lambda >= tau_1 -> m_new:active, m_old:deprecated
  - tau_0 < Lambda < tau_1 -> m_new:candidate_active (24h复核)
  - -tau_0 <= Lambda <= tau_0 -> conflicted (澄清/HITL)
  - -tau_1 < Lambda < -tau_0 -> m_old:active, m_new:candidate_hold
  - Lambda <= -tau_1 -> 拒绝新记忆
