# 配置键统一

| 配置键 | 默认值 | 说明 |
|---|---:|---|
| ENABLE_MEMORY_SERVICE | true | 启用长期记忆服务 |
| MEMORY_BASE_TAU | 0.65 | 写入阈值基线 |
| MEMORY_DELTA_EDUCATION_CORE | -0.15 | 高安全域阈值偏置 |
| MEMORY_DELTA_GENERAL_DEFAULT | 0.00 | 通用域阈值偏置 |
| MEMORY_DELTA_CASUAL_CHAT | +0.15 | 闲聊域阈值偏置 |
| MEMORY_CONFLICT_TAU0 | 0.5 | 冲突中性区间阈值 |
| MEMORY_CONFLICT_TAU1 | 2.0 | 冲突强决策阈值 |
| MEMORY_DELETE_SLA_SECONDS | 300 | 删除闭环SLA（秒） |
| MEMORY_GRAPH_RETRIEVAL_P1 | true | 图检索是否作为P1标准组件 |
| SECURITY_POLICY_MODE | strict | 安全模式（strict/trusted_local_max） |
| LOCAL_MAX_PRIVILEGE_ACK | false | 本地最高权限确认开关 |
| LOCAL_MAX_PRIVILEGE_LOOPBACK_ONLY | true | 高权限模式仅本机回环访问 |
