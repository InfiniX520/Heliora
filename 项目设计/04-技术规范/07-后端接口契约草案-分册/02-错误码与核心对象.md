## 4. 核心对象结构（首版）

### 4.1 MemoryRecord

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| memory_id | string | 是 | 记忆ID |
| user_id | string | 是 | 用户ID |
| scope | enum | 是 | global/course/project/thread |
| memory_type | enum | 是 | semantic/episodic/procedural |
| content | string | 是 | 记忆内容 |
| evidence | array | 是 | 至少1条证据 |
| importance | number | 是 | 0~1 |
| confidence | number | 是 | 0~1 |
| strength | number | 是 | 0~1 |
| status | enum | 是 | active/deprecated/conflicted/candidate_active/candidate_hold/archived/deleted |
| version | integer | 是 | 版本号 |
| lambda_value | number/null | 否 | 冲突判别值 |
| created_at | string | 是 | 创建时间 |
| updated_at | string | 是 | 更新时间 |

### 4.2 TaskStatus

| 字段 | 类型 | 说明 |
|---|---|---|
| task_id | string | 任务ID |
| status | enum | created/routed/queued/running/retrying/completed/failed/canceled |
| queue | string | realtime/normal/memory/batch |
| progress | number | 0~100 |
| note | string | 当前说明 |
| result | object | 成功结果 |
| error | object | 失败信息 |
